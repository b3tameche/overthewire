I wrote down these letters in the following order: **ETAOINSRHDLUCMFYWGPBVKXQJZ**

Also, did some "Frequency Analysis" on given strings using Python and sorted by occurrences:

```python
a = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"

text = "CGZNL YJBEN QYDLQ ZQSUQ NZCYD SNQVU BFGBK GQUQZ QSUQN UZCYD SNJDS UDCXJ ZCYDS NZQSU ..."
t = {}
for each in a:
    t[each] = text.count(each)
frequency = {a: b for a, b in sorted(t.items(), key=lambda item: item[1], reverse=True)}

print(''.join(frequency))
```

Output is rearranged alphabet:
```
SQJUBNCGDZVWMYTXKELAFIOHRP
```

Since I have little experience in ciphers, the first thing that came up to my mind was "Monoalphabetic Substitution Cipher".
We have two alphabets: **ETAOINSRHDLUCMFYWGPBVKXQJZ** and **SQJUBNCGDZVWMYTXKELAFIOHRP**, so we can assume that mapping is 
conducted between them. So I wrote the Python script to convert letters in ciphertext:
```python
freque = "SQJUBNCGDZVWMYTXKELAFIOHRP"
alphab = "ETAOINSRHDLUCMFYWGPBVKXQJZ"
cipher = "KSVVW BGSJD SVSIS VXBMN YQUUK BNWCU ANMJS"

s = ''
for each in cipher:
    s += alphab[freque.index(each)]
print(s)
```

Output:
```
WELLU IREAH ELEKE LYICN MTOOW INUSO BNCAE
```

I noticed that first 4 letters were converted correctly because it gave us the word "WELL", so every 'W','E' and 'L'
should be on correct positions. Also, there are spaces for no reason, they're just 5 letter blocks. Since I had done some
manual decryptions for monoalphabetic ciphers, I thought it would be easy to do so in this case too.

WELL...E..ELE.EL........W.........E -> dots are incorrect letters

Then I started simply guessing words, for example LE.EL should be LEVEL, so every incorrect K should be V
..E before LEVEL should be THE, so every A should be T and every H should be H:

WELL...ETHELEVEL........W........TE -> WELLDONETHELEVEL.O......WO.D.....TE -> WELLDONETHELEVEL.O......WORD...R.TE -> 
WELLDONETHELEVEL.O.RPASSWORD.S.R.TE -> WELLDONETHELEVELFOURPASSWORD.S.RUTE -> WELLDONETHELEVELFOURPASSWORDIS.RUTE

Only thing that came to my mind was BRUTE since it's cybersecurity related term. Surprisingly it worked.

Password: **BRUTE**
